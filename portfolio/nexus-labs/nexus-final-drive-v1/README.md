## Purpose

This version preserves the NEXUS visual direction and browser-app structure as a portfolio-safe version for:

- legal public crypto puzzles
- CTF research
- classical cipher practice
- toy algorithm testing
- read-only blockchain explorer lookups
- browser UI/UX portfolio demonstration
- local research notes through browser storage

## Safety / scope boundary

This version is intentionally framed for legal, public, educational puzzle work. It does not target unauthorized wallets, systems, accounts, private keys, recovery phrases, or private funds.

The Python organizer file from the ZIP is cataloged as source context only and is not autorun here.

## Run in Codespaces

```bash

```

Then open the forwarded port and visit:


