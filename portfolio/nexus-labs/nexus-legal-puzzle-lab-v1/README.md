# NEXUS — Legal Crypto Puzzle Lab v1

This is a versioned browser-app portfolio piece.

## Location

```text
portfolio/nexus-labs/nexus-legal-puzzle-lab-v1/index.html
```

## Purpose

NEXUS Legal Crypto Puzzle Lab v1 is a mobile-first browser dashboard for:

- legal public crypto puzzles
- CTF research
- classical cipher solving
- toy algorithm testing
- public blockchain education
- read-only explorer lookups
- portfolio documentation

## Safety boundary

This version is intentionally framed for legal, public, educational puzzle work. It does not target unauthorized wallets, systems, accounts, or private funds.

## Run in Codespaces

From the repository root:

```bash
python3 -m http.server 8787 --bind 0.0.0.0
```

Then open the forwarded port and visit:

```text
/portfolio/nexus-labs/nexus-legal-puzzle-lab-v1/
```
