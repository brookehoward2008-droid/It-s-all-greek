# Cleanup Report

## Summary
The original Nexus bundle mixed browser builds, Python experiments, backup archives, and utilities in one package. This GitHub publish keeps the browser-facing branch and supporting notes together.

## Included
- `index.html`
- `app/nexus-main-browser-v10.html`
- `app/nexus-winnings-v3.html`
- `launch/start_nexus_master.bat`
- documentation and inventory files

## Excluded from this GitHub publish
- large zip and 7z backups
- installers and executables
- heavy experimental branches that were not necessary for the initial repo publish

## Project type
This repository is a cleaned browser project and not an R application.
